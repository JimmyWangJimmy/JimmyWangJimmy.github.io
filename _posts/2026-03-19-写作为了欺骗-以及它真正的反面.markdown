“写作为了欺骗”这句很锋利。  
但重读后更重要的不是批判写作，而是识别“失真源”。

## 三种失真

| 失真类型 | 表现 | 代价 |
|---|---|---|
| 叙事失真 | 只写理想自我 | 无法面对真实问题 |
| 过程失真 | 只写结果不写过程 | 无法复现成长路径 |
| 行动失真 | 写完没有下一步 | 写作与生活断裂 |

## 反失真写作循环

```mermaid
flowchart LR
    A["记录真实处境"] --> B["提出可验证假设"]
    B --> C["执行小实验"]
    C --> D["反馈复盘"]
    D --> A
```

## 结论

写作不是“把自己说好”，而是“把自己看清”。  
看清之后，才谈得上改变。
